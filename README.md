# Harmony4YMM4

## 概要
HarmonyをYMM4のpluginフォルダに簡単に配置できるようにymmeにパッケージ化したもの。  
一部のYMM4プラグインの動作に、このライブラリが必要になることがあります。

## 導入方法
1. [Releases](https://github.com/tetra-te/Harmony4YMM4/releases/latest)にアクセスする。
2. `Harmony4YMM4_X.X.X.ymme`をクリックしてダウンロードする。
3. ダウンロードしたファイルをダブルクリックする。
4. 表示される画面の通りにインストールする。

> [!NOTE]
> ファイルをダブルクリックしてもインストールできない場合は、YMM4を起動して`ヘルプ(H)`>`YMM4用拡張子の関連付け`>`登録する`から拡張子の関連付けを行ってください。
> 再度ファイルをダブルクリックすることでインストール画面が表示されます。

## Harmonyについて
HarmonyはMITライセンスの下で配布されています。  
詳細については[license/Harmony.txt](license/Harmony.txt)を参照してください。

## アンインストール方法
1. YMM4を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4を終了する。
3. `Harmony`という名前のフォルダを削除する。
